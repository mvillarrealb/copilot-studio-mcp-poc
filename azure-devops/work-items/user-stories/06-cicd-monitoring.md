# Historia de Usuario: Configurar Monitoreo de Pipelines

## Épica
Integración Continua y Despliegue Continuo (CI/CD)

## Descripción

**Quiero** monitorear el rendimiento y salud de los pipelines  
**Para** identificar cuellos de botella y garantizar disponibilidad

- [ ] Alertas en caso de fallos consecutivos
- [ ] Dashboard con estadísticas de éxito/fallo
- [ ] Integración con herramientas de monitoreo

## Tareas Relacionadas
1. Configurar métricas y alertas para pipelines en Azure Monitor
2. Crear dashboard de monitoreo en Azure DevOps
